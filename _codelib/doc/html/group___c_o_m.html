Component Object Model (com)

Module

package  modDirectCOM
 

Instanz aus unregistrierter COM-dll erzeugen.


Funktionen

Function modDirectCOM.GetInstance (ByVal LibraryString As String, ByVal ProgIdString As String) As stdole.IUnknown
 Erzeugt eine Instanz aus der Klasse in einer COM-dll. Die dll muss nicht registriert sein.

Dokumentation der Funktionen

Function modDirectCOM.GetInstance ( ByVal LibraryString As  String,
ByVal ProgIdString As  String 
)

Erzeugt eine Instanz aus der Klasse in einer COM-dll. Die dll muss nicht registriert sein.

Parameter:
LibraryString Vollstaendiger Pfad zur COM-dll
ProgIdString Name der Klasse in der COM-dll
Rückgabe:
Instanz der Klasse

Beispiel:
set obj = GetInstance(CurrentProject.Path & "\EineCombibliothek.dll", "XYZ")
Damit wird eine Instanz der Klasse "XYZ" aus der dll (die sich im mdb/accdb-Verzeichnis) befindet erzeugt.
