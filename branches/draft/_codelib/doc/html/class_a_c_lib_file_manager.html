Ausführliche Beschreibung

Import/Export der Access-Objekte in/aus lokaler Code-Bibliothek.

Das Modul "_config_Application" wird unabhängig vom Import-Modus nicht mehr überschrieben, sobald es in der Anwendung vorhanden ist.

Noch zu erledigen:

Abhängigkeit zu Verweisen berücksichtigen und fehlende Verweise einstellen

getCodeLibInfoFromFile: Auslesen der CodeLib-Informationen aus Ac07-Formluaren/Berichten, die mit SaveAsText erstellt wurden (unicode-Text)
